# Tools

A collection of browser-based utility tools for daily use. All tools run entirely in the browser with no server-side processing required.

## Available Tools

### Image Tools

- **[SVG to Raster Converter](svg-to-raster-converter.html)** - Convert SVG images to PNG, JPEG, or WebP with custom dimensions, padding, and backgrounds. Supports paste, drag-drop, or URL input. Settings are saved in the URL for easy sharing.

- **[Raster to SVG Converter](raster-to-svg-converter.html)** - Convert raster images (PNG, JPEG, etc.) to SVG format.

- **[Image Compressor](image-compressor.html)** - Compress and optimize images for web use.

- **[Image Cropper](image-cropper.html)** - Crop images with precision controls.

- **[Image Compare](image-compare.html)** - Compare two images side by side with various comparison modes.

- **[Bounding Box Tool](bounding-box-tool.html)** - Draw and extract bounding box coordinates from images.

- **[EXIF Orientation Reader](exif-orientation-reader.html)** - Read and analyze EXIF orientation data from images.

### Video & Audio Tools

- **[FFmpeg Crop Generator](ffmpeg-crop-generator.html)** - Generate FFmpeg crop commands visually.

- **[Audio Visualizer](audio-visualizer.html)** - Visualize audio files with various visualization modes.

- **[YouTube Thumbnail Viewer](youtube-thumbnail-viewer.html)** - View and download YouTube video thumbnails in various resolutions.

### SVG Tools

- **[SVG Progressive Renderer](svg-progressive-renderer.html)** - Progressively render and animate SVG files.

### Social Media Tools

- **[Social Media Card Creator](social-media-card-creator.html)** - Create Open Graph and Twitter card preview images.

## Usage

Simply open any `.html` file in a modern web browser. No installation or build process required.
