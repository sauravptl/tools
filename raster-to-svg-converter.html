Base64 SVG Embedding Demo

Embed SVG images directly into HTML using base64 data URIs. This eliminates separate image files and reduces HTTP requests. SVGs embedded as images have scripts automatically disabled for security.

Security Benefit

When SVGs are embedded via <img> tags with data URIs, the browser treats them as static images. Any JavaScript, event handlers, or external references within the SVG are automatically ignored, providing safe inline image delivery.

How Base64 SVG Embedding Works

1. Original SVG

<svg xmlns="...">
  <circle cx="50" cy="50" r="40"/>
</svg>

2. Embedded as Data URI

<img src="data:image/svg+xml;base64,
PHN2ZyB4bWxucz0i..."/>
No external files Zero HTTP requests Scripts disabled Self-contained HTML
